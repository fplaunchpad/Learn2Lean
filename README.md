# Formal Verification of Elementary Arithmetic in Lean 4

A summer internship project at **FP Launchpad, IIT Madras** under **Prof. KC Sivaramakrishnan**.

## What is this?
This project formalises pen-and-paper arithmetic algorithms in Lean 4 and produces 
machine-checked correctness proofs. Not just tested on examples — mathematically guaranteed correct for all valid inputs.

## What's been verified?

- **Vertical Addition** — with carry propagation, proved correct
- **Vertical Subtraction** — with borrow propagation, proved correct
- **Long Multiplication** — in progress
- **Long Division** — in progress
- **Divisibility Tests** — digit sum rules for 3, 9, 11 — in progress
